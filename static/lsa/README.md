# Imágenes LSA

Cargá en esta carpeta las imágenes PNG del abecedario para el menú "Responder en LSA".

Los nombres deben ser simples, en mayúsculas y consistentes:

- `A.png`
- `B.png`
- `C.png`
- ...
- `Z.png`

La interfaz arma cada ruta como `/static/lsa/<LETRA>.png`. Si una imagen falta o el texto contiene un carácter no soportado, la demo muestra una advertencia y continúa sin romper la secuencia.
